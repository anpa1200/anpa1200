Hello SecurityWeek team,

I would like to share a security research/tooling release for consideration:
AdversaryGraph v4, an analyst-focused CTI-to-detection platform that now includes
a malware-analysis module and OpenCTI-ready workflows.

detection-gap analysis, STIX/OpenCTI handoff, and malware-analysis outputs in a
single browser workbench. The new v4 module adds safe static malware analysis,
decompilation/debugger-style views, clickable ATT&CK/TTP pivots, AI-assisted
function explanations, and structured dynamic-analysis summaries for analyst
review.
