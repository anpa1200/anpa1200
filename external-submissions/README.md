# External Publication Submission Kit

This folder tracks non-GitHub/non-GitLab publication and submission routes for
AdversaryGraph, MalwareGraph, AIDebug, and related CTI research.

## Current Status

| Target | Type | Asset | Status |
|---|---|---|---|
| Help Net Security | Editorial/news pitch email | `email-help-net-security.md` | Ready to send |
| SecurityWeek | Press/research pitch email | `email-securityweek.md` | Ready to send |
| Virus Bulletin | Editorial research pitch email | `email-virus-bulletin.md` | Ready to send |
| Black Hat Europe 2026 Arsenal | Tool portal | `portal-blackhat-europe-arsenal-malwaregraph-aidebug.md` | Ready for portal |
| Black Hat Europe 2026 Briefings | Research portal | `portal-blackhat-europe-briefings-adversarygraph.md` | Ready for portal |
| BleepingComputer | News tip/contact form | `form-bleepingcomputer.md` | Ready for form |
| Cybersecurity Dive | Opinion form | `form-cybersecurity-dive.md` | Ready for form |
| Hacker News / Show HN | Logged-in submission | `show-hn-adversarygraph.md` | Ready for manual post |
| Dark Reading | Commentary | `dark-reading-human-rewrite-brief.md` | Human rewrite required |

## Why Some Items Are Not Auto-Submitted

- Gmail submission is blocked until the Gmail connector is re-authenticated.
- Black Hat, BleepingComputer, Cybersecurity Dive, and Hacker News require
  portal/login/form interaction.
- Dark Reading explicitly says submitted columns must not be AI/LLM-generated,
  so only a human-authored rewrite brief is prepared.

## Recommended Order

1. Send Help Net Security, SecurityWeek, and Virus Bulletin pitches.
2. Submit Black Hat Europe Arsenal while the tool CFP is open.
3. Submit Black Hat Europe Briefings before the CFP closes.
4. Submit BleepingComputer and Cybersecurity Dive forms.
5. Post Show HN only after the public demo/docs page is stable.
